Cabeçalho

O cabeçalho de um arquivo HTML é composto por HTML, que significa o tipo de arquivo, HEAD onde se especifica como título da página e BODY que nada mais é, que a estrutura em sí da página.


Título

No Head podemos por uma tag chamada de TITLE está responsável por nomear a aba da página


Imput

Com o comando IMPUT podemos implementar diversas funcionalidades como caixas de texto, numérica ou até mesmo cores.

Como exemplo podemos ver logo abaixo


Imagem

Com o comando IMG="endereço" pode-se incluir imagens na página, essa imagens podem ser usados do caminho do próprio computador do usuário, como pode ser usada de um link direto da internet. Também é possível ajustar o tamanho da imagem com os comandos WIDTH e HEIGHT.


Tags

Existem diversas TAGS que podem ser usadas em HTML. As tags H1 e H2 por exemplo, são usadas comumente como uma forma de especificar Título e subtítulo respectivamente. já a tag P é usada para abrir um bloco de texto padrão na página, com o comando BLOCKQUOTE podemos escrever um texto em uma caixa separada, dando sensação de dica ou observação. Com a tag U, I e STRONG cria-se o efeito de sublinhado, itálido e negrito respectivamente. Com os comando tag UL, OL e LI vai ser criado listas nao ordenadas, ordenadas e listagem, respectivamente, muito parecido com algum comando que vi em Python no qual não lembro mais. o Comando HR e BR criam uma linha que separa os textos e pula uma linha em branco respectivamente. A tag MARK quando usada cria este efeito de marca texto, facilitando em trechos específicos de seu trabalho.


Links

Criando links em HTML, podemos navegar pelas páginas sendo externas ou internas, com o comando da tag a=href"endereço". Também podemos criar botões de "voltar", apenas apontando um nome à uma ID para criar o link de acesso como vemos no Índice desta página. usando # na lista pode-se criar o link direto ao ponto de ancoragem da página.
